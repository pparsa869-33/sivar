SIVAR SCHOOL WEBSITE - IMPORTANT NOTES

1) ONLINE LESSON SYSTEM
- Teachers/Admins upload lesson videos from: login.html -> admin.html
- Each lesson must have:
  - department
  - lesson title
  - teacher name
  - subject
  - video file
- After upload, the lesson appears automatically on the correct department page.

2) TEACHER ACCESS
- New users register as student by default.
- An admin must change a teacher account role to: teacher
- This can be done inside admin.html in the role management section.

3) MAP LOCATION
- The map is currently centered on Erbil, Kurdistan Region.
- If you want the exact school pin later, edit the iframe query in index.html.
